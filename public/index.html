Setup Required
The Re-Engagement Tool needs your Bullhorn credentials to connect. Follow these steps:
1
In the project folder on your Desktop, find .env.example and copy it to a new file named .env
2
Open .env in Notepad and fill in your BULLHORN_CLIENT_ID, BULLHORN_CLIENT_SECRET, BULLHORN_USERNAME, and BULLHORN_PASSWORD
3
Save the file, then restart the server: open a terminal in the project folder and run node server.js
4
Refresh this page — the tool will connect automatically
